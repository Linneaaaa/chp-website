# 📸 产品图片上传指南

## 命名规则
每张图片 = 页码文件夹 + 图片编号.jpg
例如：手册第6页第1张图 → 放入 01-round-shoulder-bottle_圆肩瓶/p6/ → 命名 p6-1.jpg

## 操作步骤
1. 打开PDF手册
2. 找到对应页（例如第6页）
3. 截图该页的产品照片
4. 重命名为 p6-1.jpg（该页第1张）、p6-2.jpg（第2张）...
5. 放入对应系列/页码文件夹

## 工厂图片
放入 images/factory/ 文件夹，名字随意即可
